# Data Dictionary

| Column Name     | Description |
|-----------------|-------------|
| **County**      | The U.S. county where the measurement was recorded |
| **Year**        | Calendar year of the measurement |
| **hw_tmax_days** | Number of heat-wave days based on daily maximum temperature (Tmax) |
| **hw_tmax_avg**  | Average number of heat-wave days based on daily maximum temperature (Tmax) |
| **hw_hi_days**   | Number of heat-wave days based on daily maximum heat index (temperature + humidity) |
| **hw_hi_avg**    | Average number of heat-wave days based on daily maximum heat index |
| **hw_nhs_days**  | Number of heat-wave days based on net heat stress (temp + humidity + radiation + wind) |
| **hw_nhs_avg**   | Average number of heat-wave days based on net heat stress |
| **Population**   | Total county population for the given year |
